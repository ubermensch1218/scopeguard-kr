# 계약서 리스크 검토표

계약서 업로드 또는 초안 작성 후 누락 조항, 모호한 조항, 위험도, 추천 보완 방향을 표로 정리하는 양식이다.

## 1. 리스크 검토

| 리스크 ID | 항목 | 탐지 기준 | 위험도 | 추천 보완 | 관련 문서 | 처리 상태 |
|---|---|---|---|---|---|---|
| RISK-001 | 착수자료 미확정 | PRD/기능정의/화면설계/검수기준 상태가 비어 있음 | 중 | 착수자료 확정표 작성 후 기능별 착수 | 계약서 / 부속서류 | 미처리 / 처리 / 제외 |
| RISK-002 | 검수 기준 부재 | 요구사항별 기대 결과와 테스트 조건이 없음 | 상 | 검수기준표와 검수실행 기록표 연결 | 계약서 / 부속서류 | 미처리 / 처리 / 제외 |
| RISK-003 | 수정횟수 기준 모호 | 수정 1회의 단위와 제출 기한이 없음 | 상 | 통합 수정요청 1세트 기준 명시 | 계약서 / 부속서류 | 미처리 / 처리 / 제외 |
| RISK-004 | 변경요청 절차 부재 | 신규 기능/화면/데이터 변경의 비용·일정 승인 조건이 없음 | 상 | 변경요청서 승인 후 착수 | 계약서 / 부속서류 | 미처리 / 처리 / 제외 |
| RISK-005 | 회의 결정 효력 모호 | 구두 요청과 회의록 승인 기준이 없음 | 중 | 회의록 승인서와 변경요청서 연결 | 계약서 / 부속서류 | 미처리 / 처리 / 제외 |
| RISK-006 | 권리귀속 모호 | 프로젝트 전용 산출물과 사전보유자산 구분이 없음 | 중 | 권리귀속 인도목록 작성 | 계약서 / 부속서류 | 미처리 / 처리 / 제외 |
