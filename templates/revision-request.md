## 분류 기준

| 분류 | 판단 기준 | 처리 방식 |
|---|---|---|
| 하자 | 계약 범위 기능 불능, 재현 가능한 오류, 검수 기준 미달 | 무상수정 회차 미차감 |
| 무상수정 | 계약 범위 내 경미한 문구, 이미지, 표시, 배치 조정 | 포함 회차 안에서 처리 |
| 변경요청 | 신규 기능, 신규 화면, 데이터 구조 변경, 외부 연동 추가, 운영정책 변경, 승인사항 번복 | 비용·일정·검수 기준 서면 승인 후 착수 |
| 제외/보류 | 자료 미제공, 기준 문서 부재, 검수와 무관한 선호 변경 | 자료 확정 또는 변경요청서 작성 후 처리 |

## 확인 사항

- [ ] 본 요청은 산출물 제출 후 계약상 제출기한 내 통합 제출되었다.
- [ ] 신규 기능, 신규 화면, 데이터 구조 변경은 포함하지 않는다.
- [ ] 승인된 PRD, 화면설계서, 디자인 방향 변경은 포함하지 않는다.
- [ ] 하자 보완 항목은 무상수정 회차에서 차감하지 않는다.
