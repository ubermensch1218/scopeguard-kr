# 22 변호사 검토 요청서

- Format: HWP
- Inputs: 프로젝트 기본 정보, 전달 패키지, 우선 검토 계약, NDA/IP 쟁점, 회신 형식
- Output: `22_변호사_검토_요청서.hwp`
- Must include: 전달 패키지, NDA/IP 집중 검토, 회신 형식
- Use when: 계약서와 부속서류를 변호사에게 검토 요청용 패키지로 보낼 때

## 작성 기준

- 계약서 본문을 대신하지 않고 검토 범위와 첨부자료를 안내한다.
- 변호사가 `21_계약서_리스크_검토표.xlsx`에 위험도, 문제, 제안 문구, 지킬 선을 회수할 수 있게 쓴다.
- NDA/IP는 실제 공개할 자료 목록, 열람자, 사전보유자산, 공동개발 산출물과 함께 묶는다.
- 법률 의견처럼 단정하지 않는다.
