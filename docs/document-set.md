# Document Set

문서 세트는 개발자와 수요자가 같은 표를 보며 합의할 수 있도록 DOCX와 XLSX를 나눠 제공한다.

스킬 구조는 최상단 orchestrator 하나와 문서별 recipe module 폴더로 나눈다. 전체 목록은 `references/documents/index.md`, 각 문서 작성법은 `references/documents/{문서번호-이름}/recipe.md`에 둔다.

## 형식 기준

| 형식 | 용도 |
|---|---|
| DOCX | 계약 본문, 설명 문구, 서명란, 사람에게 읽히는 합의 문서 |
| XLSX | 체크표, 긴 목록, 견적 row, 검수 실행, 리스크 검토처럼 행 단위로 확인할 문서 |
