# 변호사 검토 요청

scopeguard-kr은 법률 자문 도구가 아니라 SW 외주개발계약의 운영 리스크를 줄이기 위한 오픈 프로젝트입니다.

아래 항목에 대한 검토와 PR을 요청합니다.

## 검토 요청 항목

1. 하자와 경미한 수정의 경계
2. 경미한 수정과 변경요청의 경계
3. 무상수정 총 횟수와 통합 수정요청 1회 정의
4. 승인된 PRD/화면설계 변경 시 비용·일정 조정 조항
5. 회의록 승인 전 구두 발언의 효력 제한
6. 대면회의 횟수, 출장, 교육, 워크숍 비용 조항
7. 검수기준과 승인간주 조항
8. 소스코드와 수급자 사전보유자산 분리
9. 개인정보 처리위탁 및 운영비/API비 부담 조항

## PR 방식

- 계약 조항 문구 개선은 `templates/`에 제안
- 법령/지침/사례 근거는 `docs/legal-references.md`에 출처 포함
- 룰 추가는 `assets/app.js`의 `riskRules` 또는 추후 `data/risk-rules.json`에 제안

## 주의

검토 의견은 일반 정보로 공개되며, 개별 사건에 대한 법률 자문으로 표시하지 않습니다.
